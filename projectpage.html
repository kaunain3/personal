SecureIT - IoT Smart Surveillance & Access Control System

SecureIT is a distributed IoT security system combining an ESP32-based live surveillance camera with an ESP8266 smart lock. It features motion-triggered Telegram alerts, dynamic PIN-based remote authentication, and real-time MJPEG streaming. A Node.js backend powers event logging and a web dashboard for monitoring live feed and access history.

Star Classifier

Built and deployed an end-to-end machine learning API that classifies stars into six stellar types based on their physical characteristics. Trained and optimized a Support Vector Machine (SVM) using Scikit-learn, then deployed the model as a REST API with FastAPI on Render. Implemented data preprocessing, feature engineering, hyperparameter tuning, and automated inference through a production-ready API.
